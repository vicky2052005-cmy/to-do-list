# To-Do List Application

A simple and responsive To-Do List application built with HTML, CSS, and JavaScript. Users can create, complete, and delete tasks. Tasks are saved to local storage so they persist across page refreshes.

## Features

- Create new tasks with input validation
- Mark tasks as completed
- Delete tasks
- Persistent storage using localStorage
- Responsive and mobile-friendly design
- Clean, minimalist UI

## How to Run

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Start adding tasks!

No build tools, servers, or dependencies required.

## File Structure

```
todo-list/
├── index.html   # Application structure
├── style.css    # Styling and responsiveness
├── app.js       # Logic, DOM manipulation, localStorage
└── README.md    # Project documentation
```

## Usage

1. Type a task in the input field and click **Add** (or press Enter).
2. Click the checkbox next to a task to mark it as completed.
3. Click the **&times;** button to delete a task.
4. Tasks persist automatically in your browser's local storage.
